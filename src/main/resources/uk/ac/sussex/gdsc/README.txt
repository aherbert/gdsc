GDSC Plugins for ImageJ: a collection of ImageJ plugins including:

- Confined Displacement Algorithm (CDA) for colocalisation analysis
- Find Foci algorithm for foci identification (2D/3D)
- Various utility tools: 
-- Thresholding and mask generation 
-- Difference of Gaussians for contrast enhancement
-- Image/Stack alignment using correlation
